How this works

The container is a simple div and each box is just another div inside the container.

The secret for creating the grid is in the gridList.less file.

There you will find two less variables:
@grid_box_width
@grid_box_height

And most importanlty three grid layouts:
@grid.cols2
@grid.cols3
@grid.cols4
These styles use less arithmetic to specify the width of each grid layout.
The gridList.js file is responsible for binding element events (when you select one of the checkboxes above)
The only thing it does is set the container class to either list or grid (and cols2, cols3, cols4 )
